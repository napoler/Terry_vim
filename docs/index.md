一个个人用的vim编辑器配置文件


## 自动安装
适用于Debian系列系统

```
curl -s https://gitee.com/terrychan/Terry_vim/raw/master/install.sh | sudo bash
```

简短命令，托管于github。

```
curl -s https://git.io/JkMon | sudo bash
```


这里全部使用了中国的镜像，可能会有一定的延迟，不过对于中国用户在速度上有极大的提高。
